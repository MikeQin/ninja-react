## Git Clone This Repository

```
git clone https://github.com/MikeQin/ninja-react.git

# To run ContextApp
cd ninja-react/contextapp
npm install
npm start

# To run HooksApp
cd ninja-react/hooksapp
npm install
npm start
```

## Start from Scratch

### Class-based App

```
npx create-react-app contextapp
cd contextapp

npm start
```

### Hook-based App

```
npx create-react-app hooksapp
cd hooksapp

npm install uuid

npm start
```
